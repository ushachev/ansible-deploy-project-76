## Deploy Docker images with Ansible

Demo: [redmine.ushachev.site](http://redmine.ushachev.site)

### Requirements

* Ansible
* Make

### Install Ansible roles and collections

```sh
$ make install
```

### Setup servers

```sh
$ make setup
```

### Deploy application

```sh
$ make deploy
```
